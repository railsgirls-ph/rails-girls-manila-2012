## Contributing

1. Fork the project at https://github.com/railsgirls-ph/rails-girls-manila-2012
2. Install maruku
<pre class="brush: shell">
gem install maruku
</pre>
3. Make your changes
4. Test your changes
<pre class="brush: shell">
maruku Ruby.md
</pre>
5. Commit and create a pull request
